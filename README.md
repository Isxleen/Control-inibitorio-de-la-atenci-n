# Análisis de Patrones de Atención con Machine Learning

Este proyecto tiene como objetivo desarrollar un sistema para analizar y clasificar patrones de atención (control inhibitorio) utilizando datos de tareas cognitivas digitales como Stroop, Flanker y Stop-Signal.

## Objetivos

1. Procesar logs de tareas cognitivas.
2. Extraer características relevantes (tiempos de reacción, precisión, efectos de interferencia).
3. Entrenar modelos de ML para identificar perfiles atencionales.
4. Visualizar resultados e importancia de variables.

## Estructura del Proyecto

- `data/`: Datos crudos y procesados.
- `notebooks/`: Análisis exploratorio y prototipado.
- `src/`: Código fuente para procesamiento, features y modelado.
- `reports/`: Resultados y gráficos generados.
